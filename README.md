# ✅ My To Do List App

A fully functional To Do List app built with pure HTML, CSS, and JavaScript.
No frameworks. No libraries. Just the basics.

Built as part of my Full Stack Developer journey in 2026.

---

## 📸 Preview

![To Do List Preview](preview.png)

---

## ✅ Features

- Add tasks by typing and clicking Add
- Checkbox to mark tasks as done (strikethrough)
- Delete button to remove tasks
- Alert when trying to add empty task
- Clean card layout with professional styling

---

## 🛠️ Built With

| Technology | Purpose |
|-----------|---------|
| HTML | Structure — input, button, task list |
| CSS | Styling — layout, colors, flexbox |
| JavaScript | Logic — add, delete, complete tasks |

## 🧠 What I Learned

- DOM manipulation — creating elements with JavaScript
- document.createElement() to build HTML with JS
- appendChild() to add elements to the page
- li.remove() to delete elements from the page
- addEventListener for click and change events
- classList.add() to apply CSS classes via JS
- Grouping elements with div for layout control
- style.textDecoration for strikethrough effect

## 🚀 How To Run

1. Download or clone this repository
2. Open Todo.html in any browser
3. Start adding tasks

No installation needed. No dependencies. Just open and use.
